# 🎨 PROMPT POUR GÉNÉRER LE LOGO - KASED APP

**Application** : kased-app (Application de gestion des cotisations d'église)  
**Date** : 3 mai 2026

---

## 📋 Informations sur l'application

### Nom de l'application
**KASED** (prononcé "ka-sed")

### Description
Application mobile de gestion des cotisations pour les églises et communautés religieuses. Permet de :
- Gérer les membres de l'église
- Enregistrer les cultes
- Suivre les cotisations (paiements)
- Identifier les retards de paiement
- Générer des statistiques

### Public cible
- Pasteurs et responsables d'église
- Trésoriers d'église
- Secrétaires d'église
- Communautés religieuses chrétiennes

### Valeurs de l'application
- **Simplicité** : Interface claire et intuitive
- **Fiabilité** : Données sécurisées et synchronisées
- **Transparence** : Suivi clair des cotisations
- **Communauté** : Renforcer les liens dans l'église

## 🎨 PROMPT POUR DESIGNER GRAPHIQUE

### Brief créatif

**Nom de l'application** : KASED

**Type de logo** : Icône d'application mobile (app icon)

**Format** :
- Carré : 1024x1024px (haute résolution)
- Formats additionnels : 512x512px, 256x256px, 128x128px, 64x64px
- Coins arrondis pour iOS (rayon : 22.37% de la taille)
- Coins carrés pour Android

**Palette de couleurs** :
- **Primaire** : #1246C8 (Bleu royal profond)
- **Secondaire** : #FFFFFF (Blanc)
- **Accent** : #F59E0B (Ambre/Or) - optionnel
- **Neutre** : #64748B (Gris ardoise)

**Éléments visuels à considérer** :
1. **Église/Foi** :
   - Silhouette d'église simplifiée
   - Croix stylisée
   - Clocher/Steeple
   - Vitrail abstrait

2. **Gestion/Finance** :
   - Pièce de monnaie
   - Plateau d'offrande
   - Checkmark/Validation
   - Livre de comptes stylisé

3. **Communauté** :
   - Cercles représentant des personnes
   - Mains en offrande
   - Symbole d'unité

**Style** :
- Moderne et minimaliste
- Flat design avec subtiles ombres portées
- Formes géométriques simples
- Lisible à petite taille (16x16px)
- Pas de texte dans l'icône

**Inspiration** :
- Applications de gestion : Notion, Trello, Asana
- Applications religieuses : YouVersion Bible, Glorify
- Style : Flat design iOS/Android moderne

**À éviter** :
- Trop de détails (illisible à petite taille)
- Dégradés complexes
- Texte dans l'icône
- Symboles religieux trop spécifiques (rester inclusif)
- Couleurs trop vives ou agressives
