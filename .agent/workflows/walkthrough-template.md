# Compétence : Créer un Walkthrough

## Objectif

Créer un walkthrough documenté (guide pas-à-pas) pour une feature ou un processus.

## Quand l'utiliser

- Pour documenter une nouvelle feature
- Pour créer un guide utilisateur
- Pour présenter le projet

## Étapes

1. Charger le template :
   ```
   Axiom-scaffold/templates/walkthrough-template.md
   ```
2. Remplir les sections :
   - **Étapes** : Liste numérotée des actions
   - **Captures** : Screenshots ou GIFs
   - **Notes** : Remarques importantes
   - **Prérequis** : Ce qu'il faut avant de commencer
3. Sauvegarder dans :
   ```
   Axiom-scaffold/docs/walkthroughs/<feature-name>.md
   ```

## Résultat

- ✅ Guide pas-à-pas complet
- ✅ Facile à suivre
- ✅ Avec captures d'écran

## Utilisation

Idéal pour onboarding, documentation utilisateur, ou démos.
