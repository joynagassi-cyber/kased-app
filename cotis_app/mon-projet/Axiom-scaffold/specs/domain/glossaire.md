# Glossaire Métier

Ce document définit la terminologie métier utilisée dans le projet. L'agent peut s'y référer pour éviter les confusions.

## Format

Chaque terme est défini avec :
- **Définition** : explication claire
- **Synonymes** : termes équivalents
- **Antonymes** : termes opposés
- **Exemples** : cas d'usage

---

## Exemple

### Utilisateur

**Définition** : Personne qui utilise l'application.

**Synonymes** : User, Membre, Client

**Types** :
- **Utilisateur anonyme** : non authentifié
- **Utilisateur authentifié** : connecté avec un compte
- **Administrateur** : utilisateur avec privilèges élevés

**Exemples** :
- Un utilisateur peut créer un compte
- Un administrateur peut gérer les utilisateurs
